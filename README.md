## 📖 About

A hands-on UI challenge project built while following the **7 Days Flutter UI Challenge** course. This isn't about finishing in exactly 7 days — it's about dedicating focused time to deepen Flutter skills and learn how to build more attractive user interface designs.

> **Course:** 7 Days Flutter UI Challenge
> **Platform:** [BuildWithAngga](https://buildwithangga.com/kelas/course-playing/0RWMa6Vpxv)

## 🛠️ Tech Stack

| Technology | Purpose |
|---|---|
| **Flutter** | UI framework for building cross-platform apps |
| **Dart** | Programming language for Flutter |

---

## 🎯 Challenge Goals
- [ ] Understand how to design more attractive UIs in Flutter
- [ ] Implement pre-designed UI templates provided by the instructor
- [ ] Improve speed and confidence building Flutter layouts from scratch

---

## 📚 Topics Covered

- Flutter Widget fundamentals for UI building
- User Interface design principles in Flutter
- Recreating professional UI designs in code
- Layout, spacing, and visual hierarchy in Flutter

## 🚀 Getting Started

```bash
# Clone the repo
git clone https://github.com/asiata25/7-Days-of-Flutter-UI-Mastery.git
cd 7-Days-of-Flutter-UI-Mastery

# Get dependencies
flutter pub get

# Run the app
flutter run
```

> **Requirement:** Flutter SDK installed. Check with `flutter doctor` before running.
